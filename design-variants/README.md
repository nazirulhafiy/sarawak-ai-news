# Design Variants

Generated front-end directions for choosing the Sarawak AI News visual/product direction.

## A — Near-clone editorial memo
File: `variant-a-near-clone-editorial.html`

Closest to Aligned News: minimal chrome, white page, huge centered editorial headline, narrow text column, short signal blocks, quiet source list.

Choose this if the priority is: **make it feel like the inspiration first**.

## B — Sarawak-branded brief
File: `variant-b-sarawak-branded.html`

Keeps the briefing format but adds a Sarawak identity through color stripe, wider editorial layout, side thesis, and regional positioning.

Choose this if the priority is: **recognisable local identity plus editorial seriousness**.

## C — Productized intelligence brief
File: `variant-c-productized-intelligence.html`

Darker, dashboard-like, higher density. More useful as a SaaS/intelligence product, less similar to Aligned News.

Choose this if the priority is: **turning the brief into a monitoring product**.

## Current recommendation
Pick A for now. It addresses the stated complaint most directly: the current layout was not close enough to the source of inspiration.
